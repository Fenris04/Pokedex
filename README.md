# Pokédex

A modern Pokédex web application built with **HTML**, **CSS**, and **JavaScript**, using the **PokéAPI**.

## Features

- Display Pokémon cards with images, names and types
- Search Pokémon by name
- Load more Pokémon
- Detailed overlay with:
  - Base stats
  - Height
  - Weight
  - Abilities
- Responsive design
- Loading screen
- Hover effects
- Previous / Next navigation
- Type-based card colors

## API

This project uses the free PokéAPI:

https://pokeapi.co/
